About

This page is based on Grayscale. A free Bootstrap theme created by Start Bootstrap. It can be yours right now, simply download the template on the preview page. The theme is open source, and you can use it for any purpose, personal or commercial, and even modified the template.

I'm using this theme as my landing page for DEVbyCA. I modified the color and appearance to suits my taste. Grayscale basic template includes full HTML, CSS, and custom JavaScript files along with SASS and LESS files for easy customization!

This theme features two cute cat photos by @kitsanoo and @ninovisalli on Unsplash.
